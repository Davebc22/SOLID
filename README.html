Proyecto SOLID

Repositorio creado para trabajar con los principios SOLID. Este proyecto tiene como objetivo demostrar los principios SOLID a través de un conjunto de clases y lógica para el cálculo de áreas de figuras geométricas.

Comenzando

Para comenzar con el proyecto, sigue estos pasos: 1. Clona el repositorio desde GitHub. 2. Ejecuta el archivo main.py para probar las clases y principios SOLID implementados.

Uso

Puedes utilizar este proyecto como referencia para comprender y aplicar los principios SOLID en tus propios proyectos.

Contribuir
